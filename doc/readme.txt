0. Jalankan file executable
1. Buka menu New dan pilih Level / Board yang di inginkan, atau bisa juga membuat board sendiri dengan membuka menu Editor
2. Buka menu Mode dan pilih Player atau Komputer dengan strategi algoritma DFS atau BFS untuk menyelesaikan game
3. Jika memilih mode Player, gunakan klik kiri untuk melakukan drag pada pentominoes dan klik kanan untuk melakukan rotate sekaligus flip
4. Jika memilih mode Komputer, kecepatan pencarian dapat di atur pada menu Speed
5. Game akan berakhir ketika seluruh papan puzzle berhasil terisi penuh oleh seluruh pentominos yang ada

CATATAN :

- Pastikan Board sudah di set dengan memilih Level/Board pada menu New atau menu Editor sebelum memulai mode Player atau mode Komputer

- Jika ingin mengakses mode lain sementara searching masih berjalan, hentikan searching terlebih dahulu dengan menekan tombol STOP

- Terdapat fitur tambahan berupa STOP, PAUSE, dan SHOW/HIDE animation.
